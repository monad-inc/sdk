# PagerdutyEventType

EventType determines whether events are sent as 'change' or 'alert' events. We recommend reading the docs for this output before making this choice.

## Enum

* `EventTypeAlert` (value: `'alert'`)

* `EventTypeChange` (value: `'change'`)
